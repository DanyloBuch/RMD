## Тема
Побудова вкладеної навігації та оптимізація відображення великих списків у React Native із використанням `FlatList` та `SectionList`.

---

## Реалізований функціонал

* **Вкладена навігація:** * `Drawer Navigator` із кастомним боковим меню.
  * `Stack Navigator` для екранів новин і деталей.
  * Усунення подвійного header-а шляхом відокремлення Drawer і Stack header-ів.
* **Передача даних:** Передача параметрів між екранами та встановлення динамічного заголовка екрана деталей.
* **Робота зі списками (`FlatList`):**
  * Імітація Pull-to-Refresh (`refreshing`, `onRefresh`).
  * Нескінченна прокрутка Infinite Scroll (`onEndReached`, `onEndReachedThreshold`).
  * Візуальні компоненти: `ListHeaderComponent`, `ListFooterComponent`, `ItemSeparatorComponent`.
  * Оптимізація рендерингу через `initialNumToRender`, `maxToRenderPerBatch`, `windowSize`.
* **Груповані дані (`SectionList`):** Відображення списку контактів, розбитого на логічні секції із заголовками.
* **Кастомний Drawer:** Власне виїзне меню з аватаром, ПІБ, академічною групою та інтерактивними пунктами меню.
